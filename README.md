# Dataset Creator

# Setup

To setup the environment:

```
npm install phantomjs -g
```

Create empty folders:

  - `html-selected`
  - `jpeg`
  - `selected-jpeg`
  - `compiled-css-list`

To setup configuration only once:

  - `node variables-config.js`

To run creator:
  - `node index.js`
  - `node xml-configs-generator.js`
  - `phantomjs phantomjs-script.js`
